# MCP Server Development Guide

To create high-quality MCP (Model Context Protocol) servers that enable LLMs to effectively interact with external services, use this skill.

## 🚀 High-Level Workflow

### Phase 1: Deep Research and Planning
- Identify the target API/service.
- Define available tools, schemas, and authentication methods.

### Phase 2: Implementation
- Use Python or TypeScript SDKs.
- Define resources, tools, and prompts.
- Implement error handling and security.

### Phase 3: Review and Refine
- Test tool invocations.
- Ensure descriptions are clear for LLMs.

### Phase 4: Create Evaluations
- Create automated tests for the MCP tools.
