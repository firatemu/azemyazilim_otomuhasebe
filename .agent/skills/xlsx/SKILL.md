# XLSX processing

Creation, editing, and analysis of Excel files.

## Important Requirements
- **Professional Font**: Use standard fonts (Inter, Roboto, Arial).
- **CRITICAL: Use Formulas**: Use Excel formulas for calculated values instead of hardcoding.
- **Zero Formula Errors**: Verify all formulas before saving.

## Reading and Analyzing
- Use `pandas` for large data analysis.
- Use `openpyxl` for formatting and template preservation.

## Best Practices
- Preserve existing templates when updating.
- Use color coding standards for financial models.
- Implement `recalc.py` logic to verify formula consistency if possible.
