# Agent Governance Policy

This repository follows strict enterprise file discipline.

1. Temporary files (.js, .sh, debug scripts) MUST NOT be created in root, src, modules, or production directories.
2. All temporary artifacts MUST be created under:
   /tmp
   /.agent-work
3. All temporary files must include suffix:
   .generated.js
   .generated.sh
4. After task completion, all temporary artifacts MUST be deleted.
5. Permanent file creation requires explicit user instruction.
6. Never create helper scripts in production folders.
7. Never modify core architecture files unless explicitly instructed.

These rules are mandatory for all future agent executions.
